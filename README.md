# playwright-trace-cli

CLI tool that parses Playwright trace `.zip` files and outputs structured text summaries. Designed
for use with Claude Code and other AI coding assistants.

## Install

```bash
npm install -g playwright-trace-cli
```

This installs the `playwright-trace-cli` command and automatically sets up a Claude Code skill at
`~/.claude/skills/playwright-trace/SKILL.md`.

## Usage

### Summary

Get a high-level overview of a trace:

```bash
playwright-trace-cli summary trace.zip
```

```
Trace: My Test
Browser: chromium
Duration: 3.2s
Actions: 12
Errors: 1
Network requests: 45 (2 failed)
Status: FAILED
```

### Actions

List all test actions:

```bash
playwright-trace-cli actions trace.zip
```

```
#   Action                                      Duration    Status
----------------------------------------------------------------------
1   page.goto https://example.com               1240ms      ok
2   locator.click Submit                        45ms        ok
3   expect.toBeVisible                          2010ms      ERROR
     Timeout waiting for element to be visible
```

### Errors

Show only failed actions with detailed call logs:

```bash
playwright-trace-cli errors trace.zip
```

### Network

Inspect network requests:

```bash
# All requests
playwright-trace-cli network trace.zip

# Failed requests only
playwright-trace-cli network trace.zip --failed

# Requests during step #3
playwright-trace-cli network trace.zip --step 3
```

### Screenshot

Extract a screenshot from the trace:

```bash
playwright-trace-cli screenshot trace.zip --step 3 --out screenshot.png
```

### JSON Output

All commands support `--json` for machine-readable output:

```bash
playwright-trace-cli summary trace.zip --json
playwright-trace-cli actions trace.zip --json
```

## Install from Source

To install from a git checkout (for development or pre-release use):

```bash
git clone https://github.com/sminnee/playwright-trace-cli.git
cd playwright-trace-cli
npm install
npm link
```

`npm link` runs the postinstall script, which sets up the Claude Code skill at
`~/.claude/skills/playwright-trace/SKILL.md`.

## Claude Code Integration

When installed globally, a Claude Code skill is auto-installed. This lets Claude Code automatically
use `playwright-trace-cli` when analyzing Playwright test failures.

The skill teaches Claude Code to:

1. Start with `summary` to understand pass/fail status
2. Use `errors` to diagnose failures
3. Use `network --failed` to check for API issues
4. Extract screenshots for visual debugging

## Trace Format

Playwright traces are ZIP files containing:

- `trace.trace` — NDJSON file with action events (before/after pairs correlated by `callId`)
- `trace.network` — Network request data (HAR or NDJSON format)
- `resources/` — Screenshots and other binary resources

Traces are generated by running Playwright tests with `--trace on` or configuring `use.trace` in
`playwright.config.ts`.

## Dependencies

- [adm-zip](https://www.npmjs.com/package/adm-zip) — ZIP file reading
- [commander](https://www.npmjs.com/package/commander) — CLI framework

No dependency on Playwright itself.
