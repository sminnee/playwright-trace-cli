# playwright-trace-cli

CLI tool that parses Playwright trace .zip files and outputs structured text summaries.

## Dev commands

- `npm run lint` — check formatting (Prettier) and linting (ESLint)
- `npm run lint:fix` — auto-fix formatting and lint issues
- `npm run format` — format code with Prettier
- `npm test` — run tests

## Code style

- Plain JavaScript (CommonJS, not ES modules)
- Formatted with Prettier (120 char width, 100 for markdown)
- Linted with ESLint (eslint:recommended + prettier)
- 2-space indentation, LF line endings
